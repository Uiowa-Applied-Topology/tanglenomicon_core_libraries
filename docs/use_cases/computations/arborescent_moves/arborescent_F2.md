# Use Case: Computing the F2 move on a WATT

## Functionality

### Mathematical Description

```{prf:definition}
:label: UC-C-f2-D-F2


$\left(\mathrm{F}_2\right)$ Reverse the cyclic order at one vertex $v_0$, and
at each vertex at even distance from $v_0$ in the tree.
```
