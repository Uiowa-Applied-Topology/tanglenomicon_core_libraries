# Use Case: Computing the F1 move on a WATT

## Functionality

### Mathematical Description

```{prf:definition}
:label: UC-C-f1-D-F1

The _$F_1$_ move on a weighted arborescent tangle tree reverses the cyclic order
of bonds and weights at every vertex of the graph.
{cite:p}`bonahonNewGeometricSplittings2016`

```
